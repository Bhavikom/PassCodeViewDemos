com.kevalpatel.passcodeview.keys

Class KeyNamesBuilder

  • java.lang.Object
    • com.kevalpatel.passcodeview.keys.KeyNamesBuilder

  • public final class KeyNamesBuilder
extends Object
    Created by Keval Patel on 13/04/17. This is the builder class to set the names to display on each key. The keys name should be appropriate to their value. By using this you can localize the key names based on users device's current locale. "https://github.com/kevalpatel2106/PasscodeView/wiki/Add-localized-key-names"
    Author:
    kevalpatel2106
