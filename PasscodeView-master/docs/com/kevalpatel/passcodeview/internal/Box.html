    Created by Keval Patel on 09/04/17. This class defines a box inside the BasePasscodeView. The BasePasscodeView is made up of these small boxes fitted together. Some example of the implementation of this class are BoxFingerprint and BoxKeypad. Box follows the PasscodeViewLifeCycle.
